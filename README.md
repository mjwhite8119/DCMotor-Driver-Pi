# DCMotor-Driver-Pi

## I2C Setup

Full instructions are on the [Raspberry_Pi_Master_for_ESP32_I2C_SLAVE](https://github.com/MkLHX/Raspberry_Pi_Master_for_ESP32_I2C_SLAVE/tree/master) Github page.
